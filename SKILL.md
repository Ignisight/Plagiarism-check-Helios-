# Helios — Text Intelligence Skill

Helios is a two-mode text analysis engine ported from the AEGIS system. It performs:

1. **Plagiarism detection** — searches the web for matching content, scrapes source pages,
   then runs sentence-level semantic similarity + n-gram exact phrase overlap.
2. **AI content detection** — analyzes writing fingerprints (burstiness, vocabulary
   diversity, transition word density, style consistency) and combines with Claude's
   own judgment.

---

| User asks about... | Mode |
|-|-|
| Copied text, sources, originality | Plagiarism |
| ChatGPT, AI-generated, robot writing | AI Detection |
| Both / general "check this" | Run both |

## Mode 1: Plagiarism Detection

### What to do

You have access to `web_search`. Use it to find matching content on the web.

**Step 1 — Extract search queries from the text**

From the submitted text, generate up to 10 search queries:
- Take the first 10 words of the first 8–10 sentences as individual queries.
- Add 1–2 queries using rare/unusual word combinations from the text (words that appear only once and are distinctive).
- Keep queries to 8–12 words max. Do NOT use quotes in queries.

**Step 2 — Search the web**

Run the search queries in parallel using `web_search`. Collect the top 3–4 results per query. Deduplicate URLs. Aim for 8–15 unique source URLs.

**Step 3 — Analyze snippets for similarity**

For each result, you have the title, URL, and snippet. Use this to:
- Estimate **semantic similarity**: does the snippet discuss the same ideas in similar phrasing?
- Detect **exact phrase overlap**: do 4+ consecutive words from the input text appear verbatim in the snippet?
- Flag sentences from the input that closely match snippet content.

**Step 4 — Produce the report**

```
## 🕵️ Plagiarism Check Report

**Queries run:** N | **Sources found:** N | **Sentences checked:** N

### Verdict
[🚨 CRITICAL: High plagiarism risk / ⚠️ WARNING: Similarity detected / ✅ CLEAN: No significant matches]

### Source Analysis
| Source | Similarity | Phrase Overlap | Status |
|--------|-----------|----------------|--------|
| [Title](url) | High/Med/Low | Yes/No | 🚨/✅ |

### Flagged Sentences
**Sentence N** (~XX% match): *"preview of sentence..."*
↳ Source: [Title](url)

### Summary
[2–3 sentence plain-English verdict]
```

**Thresholds:**
- 🚨 CRITICAL: Multiple exact phrase matches or strong semantic match across 3+ sentences
- ⚠️ WARNING: 1–2 exact matches or moderate semantic overlap with a source
- ✅ CLEAN: No meaningful matches found

## Mode 2: AI Content Detection

Claude itself is a strong AI writing detector. Analyze the submitted text directly.

### Signals of AI-generated text

Score each signal LOW / MEDIUM / HIGH:

| Signal | AI indicator | How to assess |
|--------|-------------|---------------|
| **Burstiness** | Low variance in sentence length | Count words per sentence; compute rough std dev |
| **Vocabulary diversity (TTR)** | Low type-token ratio | Are the same words repeated often? |
| **Sentence start diversity** | Repetitive openers | Do many sentences start with "The", "This", "It", "Additionally"? |
| **Transition word density** | Overuse of: however, furthermore, moreover, additionally, consequently, therefore, nevertheless, specifically, essentially, ultimately, notably, significantly, particularly, certainly | Count occurrences |
| **Structural uniformity** | Perfectly balanced paragraphs, formulaic intro/body/conclusion | Read the overall structure |
| **Hedging language** | "It is worth noting", "It is important to consider", "This suggests that" | Scan for these phrases |
| **Confident generality** | Broad claims stated without personal experience or specific detail | Does it feel like a textbook or a person? |
| **Rhythm** | AI text has a metronomic quality — sentences feel similarly weighted | Read it aloud mentally |

### Scoring rubric

After assessing the signals, assign an overall AI probability:

- **85–100%**: 5+ strong AI signals → "Almost certainly AI-generated"
- **65–84%**: 3–4 strong signals → "Likely AI-generated"
- **40–64%**: Mixed signals → "Uncertain — could be either"
- **0–39%**: Few or no signals → "Likely human-written"

### Report format

```
## 🤖 AI Detection Report

**Words analyzed:** N | **Sentences:** N

### Verdict
[🚨 Almost certainly AI-generated / ⚠️ Likely AI / 🤔 Mixed / ✅ Likely human-written]
**AI probability: XX%**

### Writing Fingerprint
| Metric | Value | Signal |
|--------|-------|--------|
| Sentence length variance | Low/Medium/High | 🔴/🟡/🟢 |
| Vocabulary diversity | Low/Medium/High | 🔴/🟡/🟢 |
| Sentence start variety | Repetitive/Diverse | 🔴/🟢 |
| Transition word density | High/Normal | 🔴/🟢 |
| Structural uniformity | Uniform/Natural | 🔴/🟢 |
| Hedging language | Heavy/Moderate/Low | 🔴/🟡/🟢 |

### Key Observations
[2–4 bullet points citing specific phrases, patterns, or structural choices that most influenced the verdict]

### Summary
[2–3 sentence plain-English verdict]
```

---

## Important notes

- **Minimum length**: Decline gracefully if text is under 50 characters. Ask for more content.
- **Be calibrated**: AI detection is probabilistic. Always note that no tool is 100% accurate and results should be treated as indicators, not proof.
- **Be honest about limits**: If web search returns no useful results (rate limiting, unusual content), say so rather than fabricating a clean bill of health.
- **Short texts**: For texts under 100 words, note that results are less reliable due to limited signal.
- **Don't be accusatory**: Frame results as probability/risk, not as accusations. "Shows signs consistent with AI generation" not "This was written by ChatGPT."
