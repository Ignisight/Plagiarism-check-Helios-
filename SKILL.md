# Helios — Text Intelligence Skill

Helios is a two-mode text analysis engine. It performs:

1. **Plagiarism detection** — searches the web for matching content, fetches suspicious
   source pages for deep verification, then runs sentence-level semantic similarity +
   exact phrase overlap analysis.
2. **AI content detection** — analyzes 10 writing fingerprint signals to estimate
   the probability that text was AI-generated.

---

## When to use which mode

| User asks about... | Mode |
|-|-|
| Copied text, sources, originality | Plagiarism |
| ChatGPT, AI-generated, robot writing | AI Detection |
| Both / general "check this" | Run both |

## Mode 1: Plagiarism Detection

### Step 1 — Extract search queries from the text

Generate 6–8 search queries from the submitted text:
- Take the first 8–10 words of 5–6 different sentences, spread across the text.
- Add 1–2 queries using rare/distinctive word combinations unique to the text.
- Keep queries to 8–12 words. Do NOT wrap in quotes.
- Prioritize sentences that feel "finished" and specific — not transitional filler.

### Step 2 — Search the web

Run each query using `web_search`. Collect the top 3–4 results per query.
Deduplicate URLs. Target 8–15 unique source candidates.

### Step 3 — Fetch suspicious sources

For any source where the snippet shows **strong phrase overlap** (4+ consecutive
matching words) or **high semantic similarity**, fetch the full page using `web_fetch`
to confirm the match depth. This is the key verification step — snippets alone
can be misleading in both directions.

Focus fetches on the top 3–4 most suspicious sources. Use
`text_content_token_limit: 800` to keep fetches fast.

### Step 4 — Analyze matches

For each source (snippet + fetched content where available):
- **Exact phrase overlap**: 4+ consecutive words from input appearing verbatim.
- **Semantic similarity**: same ideas expressed in similar structure/phrasing.
- **Paraphrase detection**: sentence meaning is preserved but words swapped —
  still counts as plagiarism risk.
- Map which specific sentences in the input are implicated.

### Step 5 — Produce the report

```
## 🕵️ Plagiarism Check Report

**Queries run:** N | **Sources found:** N | **Pages deep-read:** N | **Sentences checked:** N

### Verdict
[🚨 CRITICAL: High plagiarism risk / ⚠️ WARNING: Similarity detected / ✅ CLEAN: No significant matches]

### Source Analysis
| Source | Similarity | Phrase Overlap | Deep Read | Status |
|--------|-----------|----------------|-----------|--------|
| [Title](url) | High/Med/Low | Yes/No | Yes/No | 🚨/⚠️/✅ |

### Flagged Sentences
**Sentence N** (~XX% match): *"first ~12 words of sentence..."*
↳ Matched in: [Title](url) — [exact phrase / paraphrase / semantic]

### Summary
[2–3 sentence plain-English verdict. Name the primary source if one dominates.]
```

**Verdict thresholds:**
- 🚨 CRITICAL: Verbatim or near-verbatim matches across 2+ sentences from one source
- ⚠️ WARNING: 1 exact match, or consistent paraphrasing from a traceable source
- ✅ CLEAN: No meaningful matches found across all sources checked

## Mode 2: AI Content Detection

Analyze the submitted text directly against 10 signals. No web search needed.

### Signals — score each LOW / MEDIUM / HIGH

| # | Signal | AI pattern | How to assess |
|---|--------|-----------|---------------|
| 1 | **Burstiness** | Low sentence-length variance | Count words per sentence; are they all 15–22 words? |
| 2 | **Vocabulary diversity** | Low type-token ratio | Same root words recycled throughout? |
| 3 | **Sentence opener variety** | Repetitive starters | Many sentences beginning with "The", "This", "It", "Additionally", "Furthermore"? |
| 4 | **Transition word density** | Overuse of connectors | Count: however, furthermore, moreover, additionally, consequently, therefore, nevertheless, notably, ultimately, specifically, essentially, particularly, certainly, importantly |
| 5 | **Structural uniformity** | Formulaic shape | Perfectly symmetrical paragraphs? Textbook intro/body/conclusion with no deviations? |
| 6 | **Hedging phrases** | Boilerplate qualifiers | "It is worth noting", "It is important to consider", "This highlights the need for", "plays a crucial role" |
| 7 | **Confident generality** | No specifics | Broad claims with no dates, names, personal experience, or concrete examples? |
| 8 | **Emotional flatness** | Even tonal register | No humor, frustration, tangents, or personality shifts — steady neutral throughout? |
| 9 | **First-person absence** | No personal voice | No "I", no opinions stated as opinions, no "in my experience"? |
| 10 | **Rhythm** | Metronomic cadence | Read mentally: do sentences feel similarly weighted, landing with the same gravity? |

### Scoring rubric

| AI Probability | Signals present | Label |
|---------------|----------------|-------|
| 85–100% | 7+ strong signals | Almost certainly AI-generated |
| 65–84% | 4–6 strong signals | Likely AI-generated |
| 40–64% | 2–3 strong signals, mixed | Uncertain — could be either |
| 0–39% | 0–1 strong signals | Likely human-written |

### Report format

```
## 🤖 AI Detection Report

**Words analyzed:** N | **Sentences:** N | **Paragraphs:** N

### Verdict
[🚨 Almost certainly AI / ⚠️ Likely AI / 🤔 Uncertain / ✅ Likely human]
**AI probability: XX%**

### Writing Fingerprint
| Signal | Observed | Strength |
|--------|----------|----------|
| Sentence length variance | Low/Medium/High | 🔴/🟡/🟢 |
| Vocabulary diversity | Low/Medium/High | 🔴/🟡/🟢 |
| Opener variety | Repetitive/Mixed/Diverse | 🔴/🟡/🟢 |
| Transition density | High/Normal/Low | 🔴/🟡/🟢 |
| Structural uniformity | Rigid/Mixed/Natural | 🔴/🟡/🟢 |
| Hedging language | Heavy/Moderate/Absent | 🔴/🟡/🟢 |
| Specificity | Generic/Mixed/Specific | 🔴/🟡/🟢 |
| Emotional register | Flat/Neutral/Variable | 🔴/🟡/🟢 |
| Personal voice | Absent/Faint/Present | 🔴/🟡/🟢 |
| Rhythm | Metronomic/Mixed/Natural | 🔴/🟡/🟢 |

### Key Observations
[3–5 bullet points quoting or describing the specific phrases/patterns that most
influenced the verdict. Be concrete — cite actual words from the text.]

### Summary
[2–3 sentence verdict. Frame as probability, not accusation.]
```

## Running both modes

When the user wants both (or says "full analysis" / "check this"), run plagiarism
first (requires web search), then AI detection. Present as two clearly labeled
sections in one response.

## Important notes

- **Minimum length**: For texts under 80 words, warn that reliability is reduced.
  Decline entirely if under 20 words.
- **Don't fabricate clean results**: If web search returns no hits, say so explicitly
  rather than issuing a false CLEAN verdict.
- **Frame carefully**: Use probability language — "shows strong signals consistent
  with AI generation" not "this was written by ChatGPT." Results are indicators,
  not proof.
- **Mixed authorship**: A human may have lightly edited AI output, or an AI may have
  been trained on human text. The fingerprint is probabilistic. Note this if signals
  are mixed.
- **Short human texts can look AI**: Formal writing (legal, academic) naturally
  suppresses personal voice and burstiness. Weigh context.
